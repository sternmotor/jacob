* Truecrypt Backup: ganzen mbr und Truecrypt Zeug sichern, see http://martin.hinner.info/fs/truecrypt/
   
	sudo dd if=/dev/sda of=/boot/truecrypt_mbrfull.dd bs=512 count=64

Sector 0 contains truecrypt loader, sectors 1-62 contain truecrypt resident boot-time decryptor (two mirrored copies), sectors 63 and 64 contain mirrored truecrypt system volume header. This header is encrypted and contains keys for system volume decryption.
